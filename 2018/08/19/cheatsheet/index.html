파이썬 cheatsheet - 지속적 업로드 예정


파이썬을 본격적으로 사용하기 시작하면서 파이썬에서 자주 쓰이는 문법에 대한 cheatsheet의 필요성을 느꼈다.

매번 사용하는 코드인데도 불구하고 막상 필요할 때는 생각이 안나는 것 때문에 검색하기에도 질렸다.

여기는 지속적으로 업데이트가 되는 나만의 python cheatsheet이 될 것이다.

  • string
  • list
  • Dictionary
  • File I/O
  • Pandas


##string

s.strip(), s.rstirp(), s.lstrip() : 공백 제거

s.replace("from","to") : 문자열 변경

s.split(char) : char기준으로 문자열 나누기, 리스트로 반환

"Today is %s" %"sunday" : 문자열 포맷팅


List

lst.append(val) : 마지막에 아이템 추가

lst.extend(seq) : 연속데이터를 마지막에 추가

lst.insert(idx, val): 인덱스 위치에 아이템 추가

lst.remove(val) : val에 해당하는 가장 첫 번째 데이터 삭제

lst.pop([idx])->value: 해당 인덱스를 pop 하여 리턴, 인덱스의 디폴트는 가장 마지막

lst.sort(), lst.reverse(): 정렬과 뒤집기 연산


Dictionary

d[key] =value : 키 추가, 이미 같은 키가 있다면 값 변경

del d[key] : 해당 키 삭제

d[key], d.get(key) : 값을 리턴, get함수를 사용하면 데이터가 없어도 오류가 발생하지 않음

d.keys(), d.values(), d.items() : 키, 값, 키와 값 리턴

d.pop(key[,default]) : 값 리턴

d.popitem() : (키, 값) 리턴


File

f=open("file.txt", "w", encoding="utf-8-sig"): open(name of file, opening mode, encoding)

opending mode: w=쓰기, r=읽기, a=append, t=text, b=binary

f.write("string") or f.writelines() : 쓰기

f.read([n]), f.readline([n]) or f.readlines() : 읽기 (문자, 한줄, 여러 줄)

f.flush(), f.tell()


Pandas

(참고 웹사이트)

dt=pd.DataFrame(Series) : 데이터프레임 선언
dt.shape(), dt.columns : 데이터 프레임 행과 열 확인
dt.unique() : 데이터가 범주형인지 확인
dt.dropna(how='any') : 결측값 제거


pandas.merge(df1, df2, on="key") : key 열을 기준으로 df1, df2에 공통적인 값이 있는 열으로만 데이터프레임 구성
pandas.merge(df1, df2, on="key", how="outer") : 위와 같이 합쳐주지만 공통적인 값이 key열에 없어도 추가하여 nan값을 가짐
pandas.merge(df1, df2, on="key", how="left"): df1의 key열을 기준으로 합침 <->how=”right”
pandas.merge(df1, df2, left_on="rkey", right_on="lkey") : 기준되는 열의 이름이 다른 경우


pandas.merge(df1, df2, left_on="key", rigth_index=True) : 왼쪽 dt의 열을 기준으로 오른쪽 dt의 인덱스값을 삽입하여 데이터프레임 구성(오른쪽 dt의 인덱스가 왼쪽 dt의 기준열의 도메인에 속해있어야 함)
pandas.merge(df1, df2, left_index=True, rigth_index=True) : 공통된 인덱스를 기준으로 데이터프레임 구성(how=”” 옵션 가능)


pd.concat(s1,s2,s3) : 시리즈를 단순 연결하여 하나의 시리즈로 연결 (s1,s2,s3는 시리즈형 데이터)
pd.concat([s1,s2,s3], axis=1, keys=["a","b","c"]) : 시리즈를 열 방향으로 연결하여 데이터 프레임 구성, keys는 열이름에 해당(해당 인덱스가 없는 경우 nan값을 가짐)
pd.concat([dt1,dt2,dt3], axis=1, keys=["a","b","c"]) : 데이터프레임의 연결도 시리즈와 동일
pd.concat([s1,s2,s3], ignore_index=True) : 인덱스를 처음부터 새로 붙여서 데이터프레임 구성
