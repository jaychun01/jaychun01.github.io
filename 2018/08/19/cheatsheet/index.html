파이썬 cheatsheet - 지속적 업로드 예정


파이썬을 본격적으로 사용하기 시작하면서 파이썬에서 자주 쓰이는 문법에 대한 cheatsheet의 필요성을 느꼈다.

매번 사용하는 코드인데도 불구하고 막상 필요할 때는 생각이 안나는 것 때문에 검색하기에도 질렸다.

여기는 지속적으로 업데이트가 되는 나만의 python cheatsheet이 될 것이다.


  • string

s.strip(), s.rstirp(), s.lstrip() : 공백 제거

s.replace("from","to") : 문자열 변경

s.split(char) : char기준으로 문자열 나누기, 리스트로 반환

"Today is %s" %"sunday" : 문자열 포맷팅


  • List

lst.append(val) : 마지막에 아이템 추가

lst.extend(seq) : 연속데이터를 마지막에 추가

lst.insert(idx, val): 인덱스 위치에 아이템 추가

lst.remove(val) : val에 해당하는 가장 첫 번째 데이터 삭제

lst.pop([idx])->value: 해당 인덱스를 pop 하여 리턴, 인덱스의 디폴트는 가장 마지막

lst.sort(), lst.reverse(): 정렬과 뒤집기 연산


  • Dictionary

d[key] =value : 키 추가, 이미 같은 키가 있다면 값 변경

del d[key] : 해당 키 삭제

d[key], d.get(key) : 값을 리턴, get함수를 사용하면 데이터가 없어도 오류가 발생하지 않음

d.keys(), d.values(), d.items() : 키, 값, 키와 값 리턴

d.pop(key[,default]) : 값 리턴

d.popitem() : (키, 값) 리턴


  • File I/O

f=open("file.txt", "w", encoding="utf-8-sig"): open(name of file, opening mode, encoding)

opending mode: w=쓰기, r=읽기, a=append, t=text, b=binary

f.write("string") or f.writelines() : 쓰기

f.read([n]), f.readline([n]) or f.readlines() : 읽기 (문자, 한줄, 여러 줄)

f.flush(), f.tell()
